HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 heading tag should only appear once on each page because it is the biggest heading.
  • The alt text can improve search engine ranking for your website.
  • Link CSS at the head element and JavaScript at the body element.

CSS

  • A margin indicated how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Assigning colors can be done using semantic references and hexadecimal code.
  • Class attribute can be used to add styles to some but not all elements.

Git

  • git status: check what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add files in the working branch to stage with -A indicating all files
  • git commit -m "descriptive messages": commit the changes in the staging with -m indicating a message with the commit
  • git pull: fetch and merge any commits from the tracking remote branch
  • git push: transmit local branch commits to the remote repo branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script, and JavaScript file is read from top to bottom.
  • An array can be used to hold a group of data with different type of data.
  • A function is a set of instructions that tells the computer how to perform a certain task, it does not automatically execute.
